# Qurating - Dataset-Scoring-and-Evaluation-System
Qurating is a system that helps evaluate and score datasets based on their quality. It is inspired by the QuRating framework and focuses on selecting better data for machine learning tasks.

It was developed as a team project by Akshit Yadav, Swastiek Kala, and Praneel Kapoor

I worked on the training pipeline and implemented the core scoring logic, where I helped design how datasets are evaluated and assigned quality scores based on defined criteria.

Project Motivation:
The idea behind the project is inspired by approaches such as *QuRating* for selecting high-quality data based on multiple criteria like writing clarity, information richness and educational value, rather than only simple heuristics.

📄 Research Context :
This project is based on the implementation of concepts from the *QuRating* research paper on dataset quality evaluation. 
As part of this project, we studied the methodology proposed in the paper and implemented a simplified version of the dataset scoring and evaluation pipeline. This work was done as a step towards understanding and exploring data quality assessment for machine learning systems.
We adapted and simplified the original approach to make it practical for smaller-scale experimentation.
